# TSB (Tuneling for Share Bytes)

![Banner](https://raw.githubusercontent.com/A01L/TSB/main/info.png)

[cite_start]**TSB** - это CLI-инструмент для **peer-to-peer** передачи файлов между удалёнными машинами[cite: 1, 18]. [cite_start]Он позволяет пересылать данные напрямую без необходимости в выделенном сервере или покупке домена[cite: 1, 17].

---

## 🛠 Как это работает

[cite_start]Инструмент использует **Flask** для создания локального сервера и **Ngrok** для проброса порта во внешнюю сеть[cite: 21, 24]. Это позволяет двум компьютерам «видеть» друг друга за пределами локальной сети.

### Основные возможности:
* **P2P передача:** Прямая связь между отправителем и получателем[cite: 1, 18].
* **Автоматизация туннеля:** Ngrok подключается автоматически при запуске режима приема[cite: 24].
* **Контроль целостности:** Использование MD5-хэша для проверки целостности переданного файла[cite: 23].
* **Прогресс-бар:** Оба клиента (отправитель и получатель) видят прогресс передачи данных в реальном времени[cite: 26].

## 🚀 Инструкция по использованию

### 1. Подготовка (единоразово)
[cite_start]При первом запуске программа запросит **Ngrok Authtoken**[cite: 22, 27]. 
* [cite_start]Получить токен можно в личном кабинете на [dashboard.ngrok.com](https://dashboard.ngrok.com/get-started/your-authtoken)[cite: 22].
* [cite_start]Токен сохраняется в файл `~/.tsb_ngrok_token` и больше не запрашивается при последующих инитах[cite: 21, 27, 28].
