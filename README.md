# TSB (Tuneling for Share Bytes)

![Banner](https://raw.githubusercontent.com/A01L/TSB/main/info.png)

**TSB** - это CLI-инструмент для **peer-to-peer** передачи файлов между удалёнными машинами. Он позволяет пересылать данные напрямую без необходимости в выделенном сервере или покупке домена.

---

## 🛠 Как это работает

Инструмент использует **Flask** для создания локального сервера и **Ngrok** для проброса порта во внешнюю сеть. Это позволяет двум компьютерам «видеть» друг друга за пределами локальной сети.

### Основные возможности:
* **P2P передача:** Прямая связь между отправителем и получателем.
* **Автоматизация туннеля:** Ngrok подключается автоматически при запуске режима приема.
* **Контроль целостности:** Использование MD5-хэша для проверки целостности переданного файла.
* **Прогресс-бар:** Оба клиента (отправитель и получатель) видят прогресс передачи данных в реальном времени.

---

## 🚀 Инструкция по использованию

### 1. Подготовка (единоразово)
* При первом запуске программа запросит **Ngrok Authtoken**. 
* Получить токен можно в личном кабинете на [dashboard.ngrok.com](https://dashboard.ngrok.com/get-started/your-authtoken).
* Токен сохраняется в файл `~/.tsb_ngrok_token` и больше не запрашивается при последующих инитах.

### 2. Получение файла
